# Customer-Shopping-Behavior-Analysis
Data Analytics | SQL | Business Insights | Strategy Recommendations

This project analyzes customer shopping behavior to uncover trends in purchasing patterns, product preferences, peak sales hours, spending behaviors, and demographic influences. The objective was to convert raw customer transaction data into actionable insights that support revenue growth, product strategy, targeted marketing, and customer segmentation.
